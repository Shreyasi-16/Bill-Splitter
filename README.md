# Bill Splitter Web Application

A simple and user-friendly web application to split group expenses fairly, calculate settlements, generate PDF reports, and share bill summaries via WhatsApp.

---

##  Features

- Create a group and add members
- Enter amount paid and purpose for each member
- Automatically calculate fair split and settlements
- Shows who should pay whom and how much
- Generate a clean, modern PDF bill summary
- Download PDF report
- Share bill summary as text directly on WhatsApp
- Fully responsive and easy-to-use UI

---

##  Tech Stack Used

- **React.js** – Frontend UI and state management  
- **Tailwind CSS** – Modern and responsive styling  
- **jsPDF & jspdf-autotable** – PDF generation  
- **WhatsApp Deep Link (`wa.me`)** – Sharing bill summary as text  

>  No database is used. All calculations are done on the client side.

---

##  How to Run the Project

1.   git clone https://github.com/Shreyasi-16/Bill-Splitter.git
2.   npm install
3.   npm run dev

##  Dependencies
npm install jspdf jspdf-autotable
